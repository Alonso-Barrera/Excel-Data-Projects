
Limpieza y Análisis de Datos en Excel
______________________________________________________________________________________
Descripción

Este proyecto consiste en la limpieza, organización y preparación de un conjunto de datos utilizando Excel, aplicando técnicas fundamentales de análisis de datos.
______________________________________________________________________________________
Limpieza de Datos

Se realizaron las siguientes acciones:

Reemplazo de valores incorrectos:
Se sustituyeron todos los valores "zero" por 0
Se reemplazó "St" por "Storage"
Manejo de valores faltantes:
Se identificaron celdas vacías en la columna Order Priority
Se rellenaron con el valor "High"
Formato de datos:
Las columnas Sales y Profit se convirtieron a formato moneda ($)
Eliminación de duplicados:
Se eliminaron registros duplicados basados en la columna Product ID
______________________________________________________________________________________
Ordenamiento y Filtros

Se aplicaron técnicas de organización de datos:

Eliminación de filtros previos
Ordenamiento con múltiples criterios:
Ship Mode (A-Z)
Order Date (más reciente a más antiguo)
Ship Date (más antiguo a más reciente)
Profit (mayor a menor)
Sales (menor a mayor)
