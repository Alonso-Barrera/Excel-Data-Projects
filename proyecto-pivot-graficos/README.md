Análisis con Tablas Dinámicas, Visualización y Macros en Excel

Descripción

Este proyecto integra el uso de tablas dinámicas, visualización de datos y automatización mediante macros en Excel para mejorar el análisis y la eficiencia en tareas repetitivas.
___________________________________________________________________________________________________________________________________________________________________________________
Tablas Dinámicas (Pivot Table)

Se creó una tabla dinámica con las siguientes características:

-Ubicación: Celda Z5
-Métrica principal: Sales
-Objetivo:
       -Analizar la participación de los clientes respecto al total de ventas
-Filtros dinámicos (Slicers):
    -Ship Mode
    -Category
    -Order Priority
___________________________________________________________________________________________________________________________________________________________________________________
Visualización de Datos

1. Tendencia de Sales
-Análisis del comportamiento de ventas a lo largo del tiempo
2. Participación de Quantity por Category
-Distribución de productos por categoría
___________________________________________________________________________________________________________________________________________________________________________________
Automatización con Macros

Se desarrollaron dos macros para automatizar tareas dentro del archivo:

*Primera Macro
-Inserta el nombre en la celda C10
-Inserta la fecha actual en la celda C12
-Aplica formato negritas y color rojo al texto en C14
-Cambia el relleno y aplica formato cursiva en C16
*Segunda Macro
-Elimina el nombre en la celda C10
-Elimina la fecha en la celda C12
-Cambia el texto a formato normal (sin negritas y color negro) en C14
-Modifica el relleno y mantiene formato en C16
