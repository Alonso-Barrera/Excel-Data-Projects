Análisis de Datos con Excel: Fórmulas, Funciones y Estadística

Descripción

Este proyecto se enfoca en el uso de fórmulas y funciones en Excel para realizar cálculos, análisis condicional y exploración estadística de datos.
____________________________________________________________________________________________________
Fórmulas y Funciones

Se resolvieron cálculos basados en la columna Sales (R) utilizando una tabla auxiliar:

-Aplicación de funciones matemáticas:

  - Suma: R + 100
  - Resta: R - V
  - Multiplicación: R × S
  - División: R / S

-Cálculos adicionales:

-Incremento del 18% sobre valores calculados
-Operaciones combinadas entre columnas
____________________________________________________________________________________________________
Cálculos Condicionales

Se aplicaron funciones lógicas como IF:

-Si State = Ontario -> R × S
  De lo contrario -> Profit

- Si Segment = Consumer y Category = Office Supplies -> Aumentar 20% a Sales
  De lo contrario -> Quantity

-Suma de Sales cuando State = Quebec

-Si Ship Mode = Standard Class o First Class -> Aumentar 18% a Profit
  De lo contrario -> 0

-Conteo de valores en Sub-Category = Binders
